# CDEV3000
AI-Governance-Efficiency-Framework (IBM Project)

## Concept: Traffic Light Classification System
A governance framework to categorize data sensitivity and AI access levels:

- **🔴 Red (Critical):** Highly sensitive data (e.g., Financial, Military). Strict prohibition or high-level clearance required.
- **🟡 Yellow (Caution):** Ambiguous or internal data. Conditional access with monitoring.
- **🟢 Green (Allowed):** Public or safe data. Unrestricted AI usage.

## Technical Prototype (Frontend)
A React-based showcase to demonstrate the classification logic interactively.
- **Stack:** React.js
