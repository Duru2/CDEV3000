# 설치 가이드 - Traffic Light AI Monitor

## Chrome/Edge 브라우저에 설치하기

### 1단계: 확장 프로그램 페이지 열기

Chrome 또는 Edge 브라우저에서 다음 주소로 이동:
```
chrome://extensions/
```

### 2단계: 개발자 모드 활성화

페이지 우측 상단의 **"개발자 모드"** 토글을 켜기

### 3단계: 확장 프로그램 로드

1. **"압축해제된 확장 프로그램을 로드합니다"** 버튼 클릭
2. 파일 탐색기에서 다음 폴더 선택:
   ```
   /home/nick/traffic-light-ai-monitor/CDEV3000/browser-extension
   ```
3. **"폴더 선택"** 클릭

### 4단계: 설치 확인

- 확장 프로그램 목록에 "Traffic Light AI Monitor" 표시 확인
- 브라우저 툴바에 신호등 아이콘 표시 확인

## 사용 시작하기

### 첫 번째 테스트

1. **ChatGPT 방문**: `chat.openai.com`으로 이동
   - 배지가 🟡 노란색으로 변경됨
   - 확장 프로그램 아이콘 클릭하여 상태 확인

2. **Google Classroom 방문**: `classroom.google.com`으로 이동
   - 배지가 🟢 초록색으로 변경됨

3. **두 탭 동시 열기**: ChatGPT와 Google Classroom을 각각 다른 탭에서 열기
   - 탭 간 전환 시 배지가 🔴 빨간색으로 변경됨
   - 위반 카운터 증가 확인

### 스누즈 기능 테스트

1. 빨간불을 5회 연속 발생시키기
2. 브라우저 알림 확인: "AI Access Blocked - Snooze activated for 10 minutes"
3. ChatGPT 접속 시도 → 경고 페이지로 리디렉션
4. 팝업에서 카운트다운 타이머 확인

### 설정 커스터마이징

1. 확장 프로그램 아이콘 클릭
2. **⚙️ Settings** 버튼 클릭
3. 원하는 설정 조정:
   - 사용자 정의 AI 웹사이트 추가
   - 스누즈 임계값 변경 (예: 3회로 변경)
   - 스누즈 시간 변경 (예: 5분으로 변경)
4. **💾 Save Settings** 클릭

## 문제 해결

### 확장 프로그램이 작동하지 않는 경우

1. `chrome://extensions/`에서 확장 프로그램 활성화 확인
2. 페이지 새로고침 (F5)
3. 브라우저 재시작

### 배지가 업데이트되지 않는 경우

1. 확장 프로그램 아이콘 클릭
2. **🔄 Refresh** 버튼 클릭

### 스누즈가 해제되지 않는 경우

1. 설정 페이지 열기
2. **🔄 Reset Current Snooze** 버튼 클릭
